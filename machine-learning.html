0. Optimiazation Algorithms

Suppose \(J\) is the cost function and \(\theta\) is the parameters of the hypothesis (model). Most supervised learning can be formulated as the optimization problem: \[ \text{arg}\min_\theta J(\theta) \]

Batch Gradient Descent

Makes corrections w.r.t. the entire dataset. 
    Repeat {
\(\theta_j := \theta_j - \alpha \dfrac{\partial}{\partial \theta_j} J(\theta)\)
    }

Stochastic Gradient Descent

Makes corrections w.r.t. each data point. 
    Randomly shuffle dataset
    Repeat {
      for i = 1:m {
\(\theta_j := \theta_j - \alpha \dfrac{\partial}{\partial \theta_j} Cost(\theta, x^{(i)}, y^{(i)})\)
      }
    }

Mini-batch Gradient Descent

Makes corrections w.r.t. batches of \(b\) data points. 
    Randomly shuffle dataset
    Repeat {
      for i = 1:b:m {
\(\theta_j := \theta_j - \alpha \dfrac{1}{b}\displaystyle\sum_{j=i}^{i+b-1}\dfrac{\partial}{\partial \theta_j} Cost(\theta, x^{(j)}, y^{(j)})\)
      }
    }

1. Linear Regression

Model

\[ h_\theta(x) = \theta^T x \]

Cost Function

\[ J(\theta) = \frac{1}{2m} \sum_{i=1}^m \Big(h_\theta(x^{(i)}) - y^{(i)} \Big)^2\]

2. Logistic Regression

Model

\[ h_\theta(x) = g(\theta^T x) \quad\text{where}\quad g(z) = \frac{1}{1+e^{-z}}\] Predict \[ y = \begin{cases} 1, \quad h_\theta(x) \geq 0.5\\ 0, \quad h_\theta(x) < 0.5 \end{cases}\]

Cost Function

\[ J(\theta) = \frac{1}{m}\sum_{i=1}^m Cost\left(h_\theta(x^{(i)}), y^{(i)}\right) \] where \[ Cost\left(h_\theta\left(x\right), y\right) = \begin{cases} -\log(h_\theta(x)) \quad\text{if } y = 1\\ -\log(1-h_\theta(x)) \quad\text{if } y = 0)\end{cases}\] With an additional regularization term, \[ J(\theta) = - \frac{1}{m}\sum_{i=1}^m \Big[y^{(i)}\log\left(h_\theta(x^{(i)})\right) + \left(1-y^{(i)}\right)\log(1-h_\theta\left(x^{(i)})\right)\Big]\\ + \frac{\lambda}{2m} \sum_{j=1}^n \theta_j^2\]

3. Support Vector Machine

Cost Function

\[ J(\theta) = - C\sum_{i=1}^m \Big[y^{(i)}cost_1(\theta^Tx^{(i)}) + \left(1-y^{(i)}\right)cost_0(\theta^Tx^{(i)})\Big] + \frac{1}{2} \sum_{j=1}^n \theta_j^2\] where \(cost_i\) is a linear function until its value reaches zero, after which it stays zero.

Optimization Objective

Since \(cost_i\) can be made zero, the optimization obejctive can be rephrased as \[ \min_\theta \frac{1}{2} \sum_{j=1}^n \theta_j^2 = \min_\theta \frac{1}{2} \lVert\theta\rVert^2 \] provided that \[ \theta^T x^{(i)} = \lVert\theta\rVert\cdot p^{(i)} \begin{cases} \geq +1, \quad\text{if } y^{(i)} = 1\\ \leq -1, \quad\text{if } y^{(i)} = 0\end{cases}\] Minimizing \(\lVert\theta\rVert\) is thus equivalent to maximizing \(p^{(i)} := \text{proj}_{\theta}x^{(i)}\), which creates a large margin.

4. Neural Networks

Model

\[ z^{(j)} = \Theta^{(j-1)}a^{(j-1)}, \quad z^{(2)} = \Theta^{(1)}x, \quad a^{(j)} = g(z^{(j)}), \quad\text{where \(g\) is non-linear} \] \[h_\Theta(x) = \Big(g \circ \Theta^{(m)} \circ \cdots \circ g \circ \Theta^{(1)}\Big)x\]

Cost Function

Suppose the network has \(L\) layers and output \(h_{\Theta}(x) \in \mathbb{R}^K \) \[ J(\Theta) = - \frac{1}{m}\sum_{i=1}^m \sum_{k=1}^K \Big[y_k^{(i)}\log\left(h_\theta(x^{(i)})_k\right) + \left(1-y_k^{(i)}\right)\log\left(1-h_\theta(x^{(i)})_k\right)\Big] \\+ \frac{\lambda}{2m} \sum_{l=1}^{L-1} \sum_{i=1}^{s_{l}} \sum_{j=1}^{s_{l+1}} \left(\Theta^{(l)}_{ji}\right)^2\]

Training Algorithm

    Randomly initialize weights
    for i = 1:m {
      Forward propagation
      Compute cost Function
      Backpropagation
    }
Backpropagation computes partial derivatives of the cost function \(\dfrac{\partial}{\partial\Theta^{(l)}_{ij}}J(\Theta)\)

5. Clustering - K-Means

Optimization Objective

\[ J\left(c^{(1)},...c^{(m)},\mu_1,...\mu_K\right) := \frac{1}{m}\sum_{i=1}^m \left\lVert x^{(i)}-\mu_{c^{(i)}}\right\rVert^2\] The two steps in the algorithm computes \(\min_{c^{(1)},...c^{(m)}}J\) and \(\min_{\mu_1,...\mu_K}J\) seperately.

Training Algorithm

    Randomly initialize K cluster centroids
    Repeat {
      for i = 1:m {
        c[i] := index of cluster centroid closest to x[i]
      }
      for k = 1:K {
        mu[k] := average of points assigned to cluster k
      }
    }

6. Dimensionality Reduction - PCA

First compute the covariance matrix \[ \Sigma := \frac{1}{m} \sum_{i=1}^n\big(x^{(i)}\big)\big(x^{(i)}\big)^T\] then project data points to the eigenvectors corresponding to the largest eigenvalues.

7. Anomaly Detection

Training

Fit distribution parameters \(\mu_i\) and \(\sigma_i\) for each feature \(x_i\) \[ \mu_j = \frac{1}{m}\sum_{i=1}^m x^{(i)}_j, \quad \sigma_j^2 = \frac{1}{m}\sum_{i=1}^m \big(x^{(i)}_j - \mu_j \big)^2 \]

Prediction

Given new example \(x\) \[ p(x) = \prod_{i=1}^n p(x_j;\mu_j, \sigma_j^2) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi\sigma_j}}\exp\left[{-\frac{(x_j-\mu_j)^2}{2\sigma_j^2}}\right]\] Predict anomaly if \(p(x) < \epsilon \)
